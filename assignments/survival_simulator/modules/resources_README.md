# 🧩 TEAM 1 — resources.py

## 📖 Що робить цей модуль

Цей модуль відповідає за **пошук їжі** на початку кожного дня.

Гравець виходить на вилазку і знаходить від 0 до 10 одиниць їжі — це випадкове число.
Знайдена їжа додається до `state["food"]`.

---

## ⚙️ Що потрібно зробити

1. Згенерувати випадкове число від 0 до 10 — це кількість знайденої їжі.
2. Додати це число до `state["food"]`.
3. Надрукувати скільки їжі знайдено.
4. Повернути оновлений `state`.

---

## 🧠 Які знання використовуються

- `import random`
- `random.randint(a, b)` — повертає випадкове ціле число від `a` до `b` включно
- `dict` — `state["food"]` зберігає кількість їжі

---

## 🔗 Як цей модуль впливає на інші

👉 Якщо тут `food` не додати (або додати 0) →
**`health.py`** зменшить здоров'я на 10 (гравець голодує).

👉 Якщо `food = 0` кілька днів поспіль →
`health` падатиме щодня → гравець може **померти**.

👉 Якщо `food > 0` →
`health.py` додасть +5 до здоров'я → гравець живе довше.

---

## 📊 Приклад

**Вхід:**
```python
state = {"health": 100, "energy": 100, "food": 0}
```

**Виконання** (random повернув 7):
```python
food_found = 7
state["food"] = 0 + 7  # = 7
print("You found food: 7")
```

**Вихід:**
```
You found food: 7
```
```python
state = {"health": 100, "energy": 100, "food": 7}
```
